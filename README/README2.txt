please execute the scripts in the following numerical order:

Phase 1: Data Engine Construction
python dataset_generator.py

Function: Generates synthetic images by blending emojis, backgrounds, and text.

Output: Populates dataset/ folder.

python split_data.py

Function: Partitions data and creates labels.

Output: Creates yolo_dataset/ structure.

Phase 2: Victim Training (The Baseline)
python train_defense.py

Function: Fine-tunes YOLOv8n on clean data.

Output: Generates V1 weights in runs/detect/emoji_defense_model.

Phase 3: Attack Simulation (The Virus)
python train_attacker.py

Function: Trains the PPO agent to find visual vulnerabilities in V1.

Output: Saves emoji_attacker_ppo_v2.zip.

Phase 4: Defense Hardening (The Vaccine)
python generate_adv_data.py

Function: Uses the trained agent to generate adversarial images.

Output: Populates yolo_dataset/adversarial/.

python retrain_defense.py

Function: Retrains the model on a mix of clean and adversarial data.

Output: Generates V2 weights in runs_v2/.

Phase 5: Final Evaluation
python evaluate_attack.py

Function: Runs a comparative stress test on both V1 and V2 models.

Output: Generates results_comparison/final_comparison_results.csv.

python generate_extra_plots.py

Function: Visualizes the metrics (Success Rate, SSIM Distribution).

Output: Saves plots to results_comparison/.
