1. Initial File
weibo_data_processed.csv: Describes preprocessed Weibo-related data (text and emoji-related raw data basis). Purpose: Serves as the initial data source for generating subsequent algorithm-derived files, specifically emoji.txt.

2. Algorithm-Generated Files

emoji.txt: Preprocessed text-emoji integrated data. Generated from weibo_data_processed.csv. Purpose: Supports vocabulary construction and model input; serves as source for emoji_processed.txt.

emoji_processed.txt: Refined text-emoji data with standardized formats. Generated from emoji.txt via core model script. Purpose: Provides high-quality training data; supports emoji position analysis.

neuron_analysis_report.json: Emoji-text correlation analysis results. Generated by NeuronAnalyzer class (based on emoji.txt data). Purpose: Supports attention reconstruction and position analysis; optimizes recommendation accuracy.

common_content_comparison_report.json: Text-emoji correlation pairs. Generated based on preprocessed text-emoji data (emoji_processed.txt). Purpose: Core data for recommendation score calculation.

simplified_position_data.json & emoji_position_stats.json (Optional): Simplified emoji position statistics. Generated from emoji_position analysis results (emoji_processed.txt + neuron_analysis_report.json). Purpose: Supports recommendation system's position probability calculation.

emoji_position_scatter_*.png: Top 5 emojis' position distribution scatter plots. Generated from emoji_processed.txt. Purpose: Visualizes emoji position rules.

emoji_comparison_scatter.png: Comparative scatter plot of top 5 emojis' positions. Generated from emoji_processed.txt. Purpose: Supports cross-emoji position rule summary.

emoji_position_dashboard.png: Comprehensive emoji position statistics dashboard. Generated from emoji position analysis results. Purpose: Facilitates quick grasp of core position rules.

recommendation_report_example_*.txt & recommendation_report_*.txt: Detailed recommendation results. Generated by intelligent recommendation system (based on correlation data and position statistics). Purpose: Presents recommendation logic.
